# v1.1.0-non-llm-complete

## Delivered Scope

- Squash-merged v1 baseline to `main`, tagged `v1.0.0-nonllm-base`.
- Expanded to fullstack monorepo with:
  - `apps/web` (Vue3 + Vite)
  - `apps/api` (Express + SQLite + session + CSRF)
  - `packages/shared-types` (shared DTO)
- Completed non-LLM targets:
  - Piano page parity enhancements.
  - Metronome page.
  - Account/password auth and comment permission flow.
  - Local recording/keymap persistence and import/export.

## Notable Features

### Piano Page
- 88-key play via mouse/touch/PC keyboard/Web MIDI.
- MIDI import/play/pause/stop/seek with metadata.
- Recorder start/stop/replay/export (`.xmid`) and local history restore.
- Comments with register/login/logout, anonymous mode, pagination/search/delete.
- Keymap profile list, preset switch, `.xkmp` import/export with validation error codes.
- Note visualization uses noteOn/noteOff duration bars.
- Soundfont local-first loading with synth fallback.

### Metronome Page
- Route `/metronome`.
- BPM + beats-per-bar + accent/normal gain.
- Start/stop controls.

### Backend
- Auth APIs: register/login/logout/me.
- CSRF token endpoint and write-request verification.
- Comment APIs: list/publish/delete with owner/admin check.
- SQLite tables: `users`, `comments`, indexed for query.
